InterNotes Web Navigator provides managed, secure access to the Internet and Web!

InterNotes Web Navigator delivers "Team Surfing" allow user to share experience of Internet within a workgroup or as individuals. Web Navigator is much more than browsing tool, it is Internet access intergrated into the core Notes environment, enabling team of users to collaborate, coordinate and communicate web content.


By applying this capability the form of @functions and LotusSccript method, Lotus gives the Notes development community opportunity to make every application a web application.


Customer are making the web navigator by:

  • Getting their favourite oline Newpaper delivered to their mailbox via a server agents every morning
  • Fetching stock prices from free stock quote servers hourly, excluding weekends
  • Worm website for "bulk-fetch" web information via server agents during low traffic period
  • Monitoring online magazines and new services
